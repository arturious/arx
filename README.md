# arx

Personal website for `arx.is-a.dev`.

## Development

```sh
npm install
npm run dev
```

## Deploy

The site is configured for GitHub Pages. Push to `main`, then enable GitHub Pages with the `GitHub Actions` source in repository settings.

## Domain

This repo includes:

- `public/CNAME` for GitHub Pages custom domain: `arx.is-a.dev`
- `is-a-dev/domains/arx.json` as the `is-a.dev` registration file

To register the domain, fork the [`is-a-dev/register`](https://github.com/is-a-dev/register) repository and add `is-a-dev/domains/arx.json` from this repo as `domains/arx.json` in the fork. After the pull request is merged, add `arx.is-a.dev` as the GitHub Pages custom domain.
